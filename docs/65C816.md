## Syntax
```naken_asm``` uses syntax that is in keeping with modern programming styles. Some traditional syntax is also available.

| Traditional    | Modern         |
|----------------|----------------|
| lda #$05       | lda #0x05      |
| lda #%10101010 | lda #10101010b |
